# VidaPlus - Backend

Simple backend for VidaPlus. Uses Node.js, Express, TypeORM and PostgreSQL.

Environment: copy `.env.example` to `.env` and set `DATABASE_URL` and `JWT_SECRET`.

Run locally:

```
cd back
npm install
npm run dev
```

After the backend starts, open:

```
http://localhost:3001/api/docs
```

Run tests:

```
npm test
```
